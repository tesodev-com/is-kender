## Message Props

The ToastMessageProps interface defines the properties required for displaying toast messages via the event bus.

```typescript
interface ToastMessageProps {
  variant?: "solid" | "outline";
  color?: "primary" | "secondary" | "success" | "info" | "warning" | "danger" | "dark";
  title?: string;
  message?: string;
  life?: number;
}
```
